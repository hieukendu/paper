# External model archives

These immutable run manifests describe the evaluated runs, including P0 multi-seed ablation, the pinned ViSoBERT baseline, and P2 low-resource runs. Model weights and adapters are deliberately not duplicated in this hand-off bundle. Use `../reproducibility/artifact_registry.json` for the source location, prediction linkage, and pinned model/archive revisions. The retained manifests support artifact-level traceability, not an independent retraining reproduction.
